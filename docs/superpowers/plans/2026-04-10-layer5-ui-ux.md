- [ ] **Step 1: Add `aria-label` and 44px touch target to `AnimatedCheckbox`**

In `apps/web/app/components/ui/animated-checkbox.tsx`:

1. Add `ariaLabel` to the interface:
```typescript
interface AnimatedCheckboxProps {
  checked: boolean
  onChange: (checked: boolean) => void
  disabled?: boolean
  size?: number
  ariaLabel?: string
}
```

2. Add `ariaLabel` to destructuring:
```typescript
export default function AnimatedCheckbox({
  checked,
  onChange,
  disabled = false,
  size = 20,
  ariaLabel,
}: AnimatedCheckboxProps) {
```

3. Update the `motion.button` style to ensure at least 44px tap target with padding:
```typescript
    <motion.button
      type="button"
      onClick={() => !disabled && onChange(!checked)}
      disabled={disabled}
      whileTap={disabled || prefersReducedMotion ? undefined : { scale: 0.9 }}
      aria-label={ariaLabel}
      aria-checked={checked}
      role="checkbox"
      style={{
        width: size,
        height: size,
        minWidth: 44,
        minHeight: 44,
        borderRadius: 6,
        border: checked ? 'none' : '2px solid var(--border)',
        backgroundColor: checked ? 'var(--accent)' : 'transparent',
        cursor: disabled ? 'not-allowed' : 'pointer',
        opacity: disabled ? 0.5 : 1,
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        flexShrink: 0,
        padding: 12,
        transition: 'background-color 0.15s ease, border-color 0.15s ease',
      }}
```

Wait ‚Äî the `size` prop controls the visual checkbox size, and padding 12px would make the inner box wrong. Better approach: use a transparent padding wrapper. Keep the visual size the same but expand the interactive hitbox:

```typescript
style={{
  width: Math.max(size, 44),
  height: Math.max(size, 44),
  borderRadius: 6,
  border: checked ? 'none' : '2px solid var(--border)',
  backgroundColor: checked ? 'var(--accent)' : 'transparent',
  cursor: disabled ? 'not-allowed' : 'pointer',
  opacity: disabled ? 0.5 : 1,
  display: 'flex',
  alignItems: 'center',
  justifyContent: 'center',
  flexShrink: 0,
  padding: 0,
  transition: 'background-color 0.15s ease, border-color 0.15s ease',
}}
```

The checkbox itself renders at `size` visually via the SVG, but the button hitbox is at least 44px.
